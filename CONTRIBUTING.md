## How to contribute

First off, thank you for considering contributing to Stos

## Areas where you can help
* **IOS UI/UX:** Improving the native feel on IOS.
* **Desktop Distribution:** Help with packaging for MSI, EXE, DMG and DEB.
* **More Features:** Help with Latex integration

## How Can I Contribute?

### Contribution Workflow
1. Fork the repository
2. Create a new branch for your feature or bugfix
3. Commit your changes with a descriptive message
4. Push your branch
5. Open a Pull Request

### Reporting Bugs
* Check the [Issues tab](https://github.com/m4ykey/Stos/issues) to see if the bug has already been reported.
* If not, open a new issue. Describe the problem, and mention your device/platform (Android, IOS, PC)

### Suggesting Enhancements
* I'm always looking for new features! Open an issue with the "enhancement" to discuss your ideas.
